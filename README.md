# 📚 Recordatorio Estudiantil

Este proyecto es una aplicación web que permite a estudiantes registrar y visualizar fechas importantes como parciales, exámenes y entregas.

## 🛠 Tecnologías usadas
- HTML
- CSS
- JavaScript
